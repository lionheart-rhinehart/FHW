# Luxury Sports Performance Wireframes

Welcome to your custom wireframe package for the luxury sports performance franchise website. This package contains three progressive wireframe fidelities following the StoryBrand framework.

## 📁 Package Contents

```
test wireframe/
├── low-fidelity-wireframe.html      - Basic structure and layout
├── mid-fidelity-wireframe.html      - Typography and content hierarchy
├── high-fidelity-wireframe.html     - Full design with colors and styling
├── content-strategy.md              - Complete StoryBrand content
└── README.md                        - This file
```

## 🎨 Wireframe Fidelities Explained

### 1. Low-Fidelity Wireframe (`low-fidelity-wireframe.html`)

**Purpose:** Show basic structure and content flow

**Characteristics:**
- Grayscale boxes and lines only
- Basic rectangles with labels
- Single font family
- Black, white, and gray colors only
- Focuses on layout hierarchy and spacing

**Best For:** 
- Initial structural discussions
- Content placement decisions
- Understanding the basic flow

---

### 2. Mid-Fidelity Wireframe (`mid-fidelity-wireframe.html`)

**Purpose:** Show content hierarchy and readability

**Characteristics:**
- Typography hierarchy with size variations
- Actual content text from StoryBrand framework
- Grayscale with minimal accent colors
- Basic spacing and padding refinement
- Shows font weight variations

**Best For:**
- Content review and approval
- Typography hierarchy feedback
- Understanding readability and flow

---

### 3. High-Fidelity Wireframe (`high-fidelity-wireframe.html`)

**Purpose:** Client-ready mockup close to final design

**Characteristics:**
- Full color palette (navy blue, electric orange, white)
- Complete luxury/energetic design aesthetic
- Google Fonts integration (Montserrat + Open Sans)
- Gradients, shadows, and visual polish
- Hover effects and transitions
- Near-production-ready design

**Best For:**
- Client presentations
- Final design approval
- Development handoff reference

1. Open wireframe in browser

## 🎯 Design Specifications

### Color Palette

The high-fidelity wireframe uses this luxury/energetic color scheme:

- **Primary Navy:** `#0a2540` (Trust, luxury, power)
- **Secondary Navy:** `#1e3a5f` (Depth variation)
- **Accent Orange:** `#ff6b35` (Energy, action)
- **Light Orange:** `#ff8c42` (Gradient variation)
- **White:** `#ffffff` (Clean, premium)
- **Light Gray:** `#f8f9fa` (Background sections)
- **Text Dark:** `#2c3e50` (Primary text)
- **Text Medium:** `#5a6c7d` (Secondary text)

### Typography

**Headings:** Montserrat (Bold, powerful, modern)
- Weights used: 300, 400, 600, 700, 900

**Body Text:** Open Sans (Clean, readable, friendly)
- Weights used: 300, 400, 600

### Design Principles Applied

✅ **Luxury:** Premium color palette, ample white space, sophisticated typography
✅ **High-End:** Gradients, shadows, polished visual effects
✅ **Clean & Powerful:** Bold headlines, clear hierarchy, confident layouts
✅ **Energetic:** Orange accents, dynamic gradients, hover animations
✅ **Friendly:** Approachable copy, welcoming imagery, balanced design

All wireframes include these 9 sections from the StoryBrand framework:

1. **Header** - Clear headline, sub-headline, primary CTA

## 💼 Client Presentation Tips

### Recommended Presentation Order:

1. **Start with Content Strategy** (`content-strategy.md`)
   - Review all copy and messaging
   - Get approval on brand voice and positioning

2. **Show Low-Fidelity First**
   - Discuss structure and content flow
   - Make any layout adjustments needed

3. **Present Mid-Fidelity Second**
   - Review typography and hierarchy
   - Confirm content readability

4. **Reveal High-Fidelity Last**
   - Showcase the full design vision
   - Discuss colors, branding, and aesthetic
   - Get final approval

### What to Say:

**Low-Fidelity:**
> "This wireframe shows the basic structure and content organization following the StoryBrand framework. It focuses on the hierarchy and flow of information without design distractions."

**Mid-Fidelity:**
> "This version adds your actual content with typography hierarchy. You can see how the headlines, body text, and calls-to-action work together to guide visitors through the story."

**High-Fidelity:**
> "This is the near-final design with the luxury sports performance aesthetic. The color palette combines powerful navy blues with energetic orange accents, creating a premium yet approachable feel that speaks to serious athletes."
